# 設計仕様

## 文書構成

- `LICENSE`: 「あるく」が権利を持つ部分に適用するMIT License全文
- `README.md`: 利用者向けのライセンス概要、第三者著作物、依存関係の案内
- 各プラグイン: 配布ファイル単体でも判別できる著作権・MIT Licenseヘッダー

## 著作権表示

新規作成部分には次の表記を使用する。

```text
Copyright (c) 2025-2026 あるく
SPDX-License-Identifier: MIT
```

派生元の著作権表示は、その由来を追跡できるよう既存表記を維持する。

## 権利範囲

MIT Licenseはソースコードと、このリポジトリで独自作成した文書に適用する。スクリーンショットに表示されるRPGツクールMZ標準素材、依存プラグイン、派生元著作物の権利を上書きしない。

## 検証方針

- JavaScriptは `node --check` で構文を検証する。
- テキストファイルはUTF-8として読み取れることを検証する。
- `rg` で著作権表記、SPDX識別子、ライセンスリンクを確認する。
- `git diff --check` で空白エラーを確認する。
